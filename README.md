# Samantha X Monitor

X (Twitter) 自媒体运营自动化工具。

## 功能

- 🔍 自动监控关注的 X 账号
- 🧠 AI 分析形成独立观点
- ✍️ 生成落地原创推文
- 🚀 自动发布到 X

## 技术栈

| 组件 | 技术 |
|------|------|
| 抓取 | twikit |
| 发布 | baoyu-post-to-x |
| 调度 | OpenClaw Cron |

## 使用

```bash
# 配置 Cookie 后运行
./x-monitor.sh
```

## 注意

- 需要维护 X Cookie
- 账号列表本地维护，不公开
